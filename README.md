# Iac Mackenzie Cloud multi-region


## Atividade
Criar. um módulo terraform que crie em multi regiões (2) os seguintes recursos:

 - VPC
 - 3 subnets
 - Internet Gateway
 - 2 máquinas EC2
 - 1 banco de dados RDS PostgreSQL
 - 1 balanceador de carga
 - Security Group para cada recurso acima
 
 ## Tempo
 
60 minutos
